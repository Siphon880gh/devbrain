
If you want the **absolute minimum** needed to create a child theme, this is it.

### File structure

```
/wp-content/themes/your-child/
├── style.css
└── functions.php
```

### style.css

```css
/*
 Theme Name: Your Theme Child
 Template:   parent-theme-folder-name
*/
```

Again:  
`Template:` must equal the **parent theme folder**, not the theme’s marketing name.

### functions.php

```php
<?php
add_action( 'wp_enqueue_scripts', function () {
    // Load parent CSS if it doesn't already enqueue itself
    wp_enqueue_style(
        'parent-style',
        get_parent_theme_file_uri( 'style.css' )
    );

    // Always load the child stylesheet
    wp_enqueue_style(
        'child-style',
        get_stylesheet_uri(),
        ['parent-style']
    );
}, 20);
```

That’s enough for WordPress to:

- Register the child theme
    
- Inherit the parent theme
    
- Load your CSS safely
    

From here, everything else is optional.

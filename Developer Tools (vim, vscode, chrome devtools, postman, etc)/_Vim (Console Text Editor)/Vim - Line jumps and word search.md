## Line jumps with more `:` and `/`

To jump to a line 42:
```
:42
```

To search for a word "fox":
- Type `/` in place of `:`. It'll be taken as a typed command as well
```
/fox
```

- Then press n to go to next match. Press shift+n to go to previous match.

Want to search then substitute words (aka find and replace)? Refer to [[Vim - Search then substitute words (aka find and replace)]]

Newly released in 2024
https://developer.mozilla.org/en-US/docs/Web/CSS/Reference/Values/color_value/light-dark


```
body {
	background-color: light-dark(#fff, #444);
	color: light-dark(#444, #fff);
}
```

With fewer lines of css code, you can set light and dark mode colors. 

However, on older web browsers this won't work.

You should know by now:
- Tailscale is easy to install
- Login through webpage. Terminal doesnt need to open the webpage. It gives you a URL to login
- The device tailscale is on will get assigned its own IP address
- You can easily change the IP address. 
- **All ports** on that IP are opened **as if visiting the localhost** on that machine
	- localhost:3000 -> Tailscale_IP:3000

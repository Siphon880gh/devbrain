- **Check propagation from multiple locations.** You can use [WhatsMyDNS](https://whatsmydns.net/) to see whether your domain is already resolving through Cloudflare in different regions. If you use a VPN that lets you switch locations (like  [VPN Unlimited](https://id.keepsolid.com/invite/vpn/uKx0iM6pLnHan)), you can test from a region that already shows propagation. This helps confirm that the domain still opens correctly and that your web server or vhost is catching the domain and serving the site as expected.
- **Verify with `dig` from your terminal.** Once propagation has reached your location (or your VPN location), you can run a `dig` command locally to confirm that the domain is resolving to Cloudflare’s proxy IPs rather than exposing your origin server’s real IP address. Dig command can be `dig yourdomain.com`.

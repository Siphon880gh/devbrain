## The 7 layers of security

**1. Physical security**  
This is protection of the actual building, room, devices, and hardware.  
Examples include locks, badge access, security cameras, guards, and secured server rooms.  
If someone can physically access a machine, they may bypass many digital protections.

**2. Perimeter security**  
This layer protects the edge of your environment from outside traffic.  
Examples include firewalls, web application firewalls, gateway filters, and DDoS protection.  
Its job is to block unwanted traffic before it gets deeper into your systems.

**3. Network security**  
This focuses on controlling movement inside the network.  
Examples include internal firewalls, VLANs, segmentation, intrusion detection systems, and access control lists.  
Even if an attacker gets in, network security helps stop them from moving from one system to another.

**4. Endpoint security**  
Endpoints are devices like laptops, desktops, phones, and servers.  
Examples include antivirus, EDR, patching, disk encryption, device hardening, and least-privilege settings.  
Since many attacks land on a device first, this layer is critical.

**5. Application security**  
This protects the software and websites people use.  
Examples include secure coding, input validation, authentication controls, session security, dependency updates, and vulnerability testing.  
This layer helps prevent common issues like SQL injection, XSS, broken auth, and insecure APIs.

**6. Data security**  
This protects the actual information, whether stored or in transit.  
Examples include encryption, backups, data classification, access controls, tokenization, and data loss prevention.  
The goal is to keep sensitive data safe even if another layer is breached.

**7. User security**  
People are often the most targeted layer.  
Examples include security awareness training, phishing resistance, strong passwords, MFA, and safe access practices.  
A well-trained user can stop an attack before it spreads.

## Why layered security matters

A business should not depend on only one defense, like a firewall or antivirus. Attackers look for the weakest point. Layered security reduces that risk by creating multiple checkpoints.

For example:

- A phishing email may bypass one filter.
    
- MFA may still stop the login.
    
- Endpoint protection may stop malware from running.
    
- Network segmentation may stop lateral movement.
    
- Backups may help recovery if ransomware hits.
    

## Simple summary

The 7 layers of security are:

1. Physical
    
2. Perimeter
    
3. Network
    
4. Endpoint
    
5. Application
    
6. Data
    
7. User
    

Together, they create a stronger, more resilient security strategy.

Here are common memory size conversions:
```
1024M = about 1 GB
2048M = about 2 GB
512M  = about 0.5 GB
768M  = about 0.75 GB

```

It is more related to **RAM/memory sizing**, not CPU stats.

In computing, memory is often thought of in powers of 2:
```
1024 KB = 1 MB
1024 MB = 1 GB
1024 GB = 1 TB
```


So when you see:
```
memory_limit = 1024M
```


you can think:
```
Each PHP request can use up to about 1 GB of RAM.
```


**VNC** is one type of remote desktop technology. Mac has built-in VNC-style screen sharing, but it can be slow. The benefit is that you can connect to your Mac desktop from another device, even from your iPhone.

**RDP** means **Remote Desktop Protocol**. It is Microsoft’s remote desktop technology and is commonly used to control Windows computers remotely.

**Chrome Remote Desktop** is Google’s remote desktop technology. It uses a browser or mobile app as the viewer, but the computer being controlled runs a Chrome Remote Desktop Host service. This lets you control the full Windows, Mac, or Linux desktop, not just the Chrome browser.

**HTML5 remote desktop** is a different technology approach. It usually means the remote desktop viewer runs inside a web browser, so it can be cross-platform. As long as the device has a modern browser, it can often connect from Mac, Windows, Linux, Chromebook, iPhone, iPad, or Android.

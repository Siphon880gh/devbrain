
SSH tunneling = Connecting the remote port to your local port so that you can visit that remote port (eg. web portal) at your local address like http://localhost:XXXX

---

Local port forwarding a.k.a. SSH Tunneling

You may notice the term "local port forwarding" when reading documentations for the SSH tunnel command because in the command `ssh -l` that makes SSH tunneling possible, the `l` stands for local port forwarding.

### Create a `.tar.gz`

```bash
tar -czf archive.tar.gz folder-name/
```

### Extract a `.tar.gz`

```bash
tar -xzf archive.tar.gz
```

### Create a `.tar`

```bash
tar -cf archive.tar folder-name/
```

### Extract a `.tar`

```bash
tar -xf archive.tar
```

### List contents without extracting

For tar:

```bash
tar -tf archive.tar
```

For tar.gz:

```bash
tar -tzf archive.tar.gz
```

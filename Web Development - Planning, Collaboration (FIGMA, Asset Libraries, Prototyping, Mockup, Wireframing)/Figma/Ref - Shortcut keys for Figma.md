Figma has many keyboard shortcuts that make designing faster.

You do not need to memorize all of them at once. Start with the basics: move, frame, text, shape tools, zoom, comments, and prototype mode. Over time, the shortcuts become muscle memory.

Most of the shortcuts below are Mac-focused. On Windows, many `⌘` shortcuts usually map to `Ctrl`, and many `⌥` shortcuts map to `Alt`.

## Essential Shortcuts

These are useful shortcuts to learn first.

|Action|Shortcut|
|---|---|
|Show / Hide UI|`⌘ \`|
|Pick color|`⌃ C`|
|Quick actions|`⌘ /` or `⌘ P`|

Quick actions are especially useful because they let you search for commands inside Figma without digging through menus.

These shortcuts help you control how you view and inspect a design.

|Action|Shortcut|
|---|---|
|Rulers|`⇧ R`|

|Zoom out|`-`|
|Zoom to 100%|`⇧ 0`|

## Editing Modes: Design vs Prototype

Figma has different editing modes.

**Design mode** is where you create the interface. This includes frames, text, colors, components, shapes, spacing, layout, and visual design.

**Prototype mode** is where you connect screens and define interactions. This is where you create clickable flows, transitions, overlays, and user journeys.

|Action|Shortcut|
|---|---|
|Switch between Design and Prototype mode|`⇧ E`|

This shortcut is useful when you are quickly switching between designing screens and wiring them into a prototype.

## Presentation Mode

Presentation mode lets you run the design like a clickable prototype.

|Platform|Shortcut|
|---|---|
|Mac|`⌘ Cmd + ⌥ Opt + Return`|
|Windows|`Ctrl + Alt + Enter`|

Use presentation mode when you want to test the flow yourself, demo a prototype, or show the design to a teammate or customer.

These shortcuts are helpful when switching between building the design, managing components, creating prototype connections, and inspecting details for development.

## Final Takeaway

Figma shortcuts are not just about speed. They also make the design process feel smoother.

Start with the shortcuts you use every day, then slowly add more as your workflow grows. The more comfortable you become with shortcuts, the easier it is to stay focused on the design instead of searching through menus.

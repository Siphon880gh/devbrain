
## TUI: Text User Interface

**TUI** stands for **Text User Interface**.

It is a user interface that runs inside the terminal, but it is more visual and interactive than plain command-line commands.

For example, instead of typing one command, getting output, and typing another command, a TUI may let you move around with arrow keys, select menu items, scroll through panels, or watch live status updates inside the terminal.

TUI is not a new term. Linux and Unix users have used TUIs for a long time. Tools like terminal file managers, text editors, server dashboards, package managers, and system monitors often use this style.

Common examples include:
- top
- htop
- nano
- vim
- nmtui

These are not full desktop apps with buttons and windows like a GUI, but they are also not simple one-line command tools. They sit in the middle: visual, keyboard-driven, and terminal-based.

In 2026, the term **TUI** started getting more attention again because tools like **OpenClaw** include an **OpenClaw TUI**. That means OpenClaw has a terminal-based interface where you can watch or interact with the system without needing a full web dashboard or desktop app.

So when you see **TUI**, think:

> “A terminal app with a visual interface.”

It is an old Linux term, but newer AI and developer tools are making it more visible again.

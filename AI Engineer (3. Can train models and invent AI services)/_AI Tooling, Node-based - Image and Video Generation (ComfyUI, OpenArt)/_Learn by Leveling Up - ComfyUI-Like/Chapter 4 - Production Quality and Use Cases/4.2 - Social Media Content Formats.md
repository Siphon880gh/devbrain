## Learning objectives

- Generate **aspect-ratio sets** from one workflow (1:1, 4:5, 9:16).
- Plan **carousel** and **cover frame** assets in one session.
- Keep visual brand consistency (LoRA + palette words).

---

## Format map

| Format | Aspect | Use |
|--------|--------|-----|
| Feed square | 1:1 | Instagram, LinkedIn |
| Feed portrait | 4:5 | Instagram preferred |
| Story/Reels cover | 9:16 | Vertical hook frame |
| Thumbnail | 16:9 | YouTube, X video |

---

## Batch workflow

1. Define **one core prompt** + LoRA from Chapter 2.
2. Duplicate latent sizes or use Empty Latent per ratio.
3. Lock seed where composition should feel related.
4. Generate **cover frame** with bold subject and negative space for text overlay (add text in CapCut/Canva — not in-gen).
5. Upscale only finals ([[4.0 - Upscaling and Resolution Workflows]]).

---

## Brand consistency tokens

Pick 3 recurring prompt tokens:

- Lighting: `golden hour rim light`
- Palette: `teal and orange grade`
- Lens: `35mm cinematic`

Reuse across posts so feed feels intentional.

---

## Beginner mistakes

- Text baked into image — illegible at thumbnail size
- Different LoRA every post — no channel identity
- Skipping safe margins — faces cropped by platform UI

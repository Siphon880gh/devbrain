**Platform note:** Stable for most computers — ESRGAN 4x; **Ultimate SD Upscale** may need more VRAM or RunComfy.

---

## Learning objectives

- Chain **latent upscale** vs **pixel upscale** (ESRGAN, UltraSharp).
- Use tile-based upscale for large prints without OOM.
- Deliver images that survive Instagram/TikTok recompression.

---

## Upscale types

| Method | How it works | Best for |
|--------|--------------|----------|
| **Latent upscale** | Upscale in latent space + KSampler refine | Adding detail at 2× |
| **ESRGAN / model upscale** | Pixel upscaler model | Fast 4× size bump |
| **Ultimate SD Upscale** | Tiled KSampler over upscaled image | Hero stills, print |

---

## Steps

1. Start from Chapter 3 best portrait or product still.
2. Load simple **Upscale Model Loader → Image Upscale with Model** (4x).
3. Compare file size and sharpness at 100% zoom.
4. If template available, run **Ultimate SD Upscale** at 1.5–2× with low denoise.
5. Export PNG for archive; export JPG 85–92 quality for social.

---

## Social-safe resolutions

| Platform | Target long edge |
|----------|------------------|
| Instagram feed | 1080–1350 px |
| Stories/Reels thumb | 1080×1920 |
| X/Twitter | 1200–1600 px wide |

Upscale **then** crop to aspect — not the reverse.

---

## Beginner mistakes

- Upscale before fixing face/hands — magnifies errors
- 4× on already soft image — mush; fix base gen first
- Ultimate SD Upscale full image no tiles on 8GB VRAM

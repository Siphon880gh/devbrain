## Learning objectives

- Run structured sweeps for seed, steps, CFG, and sampler.
- Know typical ranges for SDXL vs Flux.
- Build a personal “default preset” per checkpoint.

---

## Parameter guide

### Seed

- `-1` = random each run
- Fixed seed = reproducible (same prompt + settings)
- **Practice:** pick best of 8 random seeds, then lock it

### Steps

| Model | Typical range |
|-------|---------------|
| SDXL | 20–30 |
| Flux Dev | 20–28 |
| Flux Schnell / turbo | 4–12 |

More steps ≠ always better past model-specific knee.

### CFG (guidance)

| Model | Typical range |
|-------|---------------|
| SDXL | 6–9 |
| Flux | 3.5–5 (often lower than SDXL) |

Too high → harsh contrast, artifact halos.

### Sampler + scheduler

- Start with template defaults
- `dpmpp_2m` + `karras` common on SDXL
- Flux templates often ship tuned pairs — change one at a time

---

## Sweep protocol (one hour)

Use one prompt and locked latent size.

1. **Seed sweep:** 8 runs, `-1` each → pick winner → lock
2. **CFG sweep:** winner seed, CFG 4 / 6 / 8 / 10
3. **Steps sweep:** ±4 from default
4. **Sampler:** 2 alternatives only

Log results in a simple table (seed, CFG, notes).

---

## Beginner mistakes

- Sweeps while also editing prompt text
- Applying SDXL CFG habits to Flux
- Ignoring scheduler when changing sampler

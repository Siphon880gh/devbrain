## Learning objectives

- Apply **face detailer** pass on portraits without redoing full image.
- Fix hands via inpaint or dedicated hand fix nodes when available.
- Know when upscale + detailer beats re-sampling entire scene.

---

## Face detailer pattern

```
Base image → Detect face bbox → Crop → KSampler refine → Paste back
```

Often wired as **FaceDetailer** or similar custom node group.

### Steps

1. Load portrait with visible face artifact.
2. Add face detailer template (search Comfy.org / RunComfy “face detail”).
3. Install missing nodes via Manager if local.
4. Run with conservative denoise on face crop (0.3–0.5).
5. Compare full regen vs detailer — detailer usually faster.

## Hand fixes

Hands remain a weak spot:

| Approach | When |
|----------|------|
| **Inpaint mask on hands** | Wrong finger count |
| **ControlNet OpenPose** | Prevent issue upstream |
| **Negative prompt** | `extra fingers, missing fingers` — partial help |
| **Reference pose** | Complex hand poses |

Workflow: generate → mask hands → inpaint with `natural hands, five fingers, anatomically correct`.

---

## Quality order of operations

1. Good base composition (ControlNet / prompt)
2. Face detailer if needed
3. Hand inpaint if needed
4. Upscale (Chapter 4)

---

## Beginner mistakes

- Face detailer denoise too high → different person
- Detailer before fixing obvious prompt/LoRA conflict
- Expecting 100% hand accuracy without pose guide

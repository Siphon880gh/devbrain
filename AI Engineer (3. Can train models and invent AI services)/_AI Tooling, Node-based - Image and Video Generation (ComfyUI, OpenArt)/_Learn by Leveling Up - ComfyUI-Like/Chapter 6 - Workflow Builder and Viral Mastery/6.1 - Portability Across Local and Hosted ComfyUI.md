## Learning objectives

- Export workflow JSON from local ComfyUI and import on RunComfy (or reverse).
- Resolve **missing checkpoint** and **missing node** errors on import.
- Maintain a **model manifest** for your signature workflows.

---

## Export / import steps

### Local → RunComfy

1. **Save** workflow (API format or standard workflow JSON).
2. Start RunComfy GPU instance.
3. **Load** JSON — read missing node list.
4. Install missing custom nodes via host UI or pick equivalent template nodes.
5. Upload or select hosted checkpoints matching your manifest filenames.
6. Queue test at low resolution.

### RunComfy → Local

1. Download JSON + note all model filenames.
2. Install custom nodes locally via ComfyUI Manager.
3. Download weights to correct folders.
4. Load — fix path differences if any.

## Compatibility tips

- Prefer **core nodes only** in portable base workflows
- Isolate exotic nodes in optional subgraph you can strip for travel
- Same ComfyUI major version reduces JSON friction

---

## Beginner mistakes

- Assuming RunComfy has your exact Civitai filename
- Porting workflow without noting custom node pack names
- Paying for GPU time during long model downloads — prep list first

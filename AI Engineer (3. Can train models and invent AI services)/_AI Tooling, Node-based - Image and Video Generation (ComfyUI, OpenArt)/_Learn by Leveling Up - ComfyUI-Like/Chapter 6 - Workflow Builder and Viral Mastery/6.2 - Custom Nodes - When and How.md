**Platform note:** Dependency-resolved — installing custom nodes changes your environment; snapshot before bulk installs.

---

## Learning objectives

- Decide **when** a custom node is worth it vs core nodes only.
- Install via **ComfyUI Manager** safely.
- Debug “node not found” after git pull or update.

---

## When you need custom nodes

| Need | Example node packs |
|------|-------------------|
| Face detail | Impact Pack, face detailer workflows |
| Advanced ControlNet | ControlNet auxiliary preprocessors |
| Video | AnimateDiff, SVD helper nodes |
| Utility | ComfyUI-Manager itself, image tools |

**Rule:** If Comfy.org template lists >3 custom packs you do not have, evaluate RunComfy before installing 20 repos locally.

---

## Install steps (local)

1. Open **ComfyUI Manager**
2. **Install Custom Nodes** → search exact name from workflow error
3. Restart ComfyUI
4. **Install Missing** if workflow loader offers it
5. Re-queue workflow

---

## Safety habits

- Install **one pack at a time** when debugging
- Note git URL in your manifest ([[6.1 - Portability Across Local and Hosted ComfyUI]])
- After ComfyUI core update, retest signature workflow

---

## Promptus note

Template platforms hide most custom nodes — when you outgrow them, you are ready for this lesson and native ComfyUI.

---

## Beginner mistakes

- Installing every trending pack — dependency hell
- No restart after install — false “still missing” errors
- Custom node tutorial from 2023 on ComfyUI 2026 paths

---

## Project checkpoint

**One custom node pack** installed for a purpose (e.g. face detail), used once, documented in manifest.

## Learning objectives

- Choose the right platform before each lesson using a short checklist.
- Know when Promptus beats local setup and when RunComfy is worth the cost.
- Avoid spending hours on dependency fixes when cloud is faster.

---

## Quick decision tree

```
Does the lesson say "Stable for most computers"?
  YES → Try local ComfyUI or Promptus first
  NO → Continue

Does it require CUDA or heavy custom nodes?
  YES → Use RunComfy (or local NVIDIA GPU)
  NO → Try local with ComfyUI Manager; escalate to RunComfy if stuck > 1 hour

Just browsing workflows for ideas?
  YES → Comfy.org / OpenArt only — no GPU
```

---

## When to use each platform

| Situation | Best choice |
|-----------|-------------|
| Never installed ComfyUI; want first image today | **Promptus.ai** template |
| M1 Mac or modest GPU; txt2img basics | **Local ComfyUI** |
| Missing nodes, CUDA-only, video, big models | **RunComfy** |
| Learning workflow structure without running | **Comfy.org** browse |
| Prompt and style research | **OpenArt.ai** |
| Tired of dependency conflicts | **RunComfy** (time vs money trade) |

---

## Local setup expectations

**Works well locally (typical):**

- SDXL / Flux Schnell txt2img
- Basic img2img
- Simple LoRA loads

**Often painful locally:**

- Latest video stacks with many custom nodes
- Workflows listing 10+ missing nodes
- CUDA-only optimizations on Mac

Use **Stability Matrix** or **ComfyUI Manager** for installs. If a tutorial references moved folders or renamed nodes, check the workflow’s post date — ComfyUI changes quickly.

---

## RunComfy cost awareness (2026 ballpark)

- Subscription often around **$30/month** with included credits.
- GPU time often around **$1/hour** depending on tier.
- **Models downloaded on the server may disappear when the instance stops** — note what you wget or upload.
- Set spending limits (some users use virtual cards) if you forget to stop instances.

## Promptus vs native ComfyUI

| | Promptus | Native ComfyUI |
|---|----------|----------------|
| Learning curve | Lower | Higher |
| Flexibility | Template-bound | Full graph |
| Custom nodes | Limited | Full |
| Best for | Ch. 1 first wins | Ch. 2–6 mastery |

Many learners start on Promptus for [[1.1 - Your First Generated Image from a Template]], then move to local or RunComfy by Chapter 2.

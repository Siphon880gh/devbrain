## Learning objectives

- See the full learning path from first template run to original viral-ready workflows.
- Know which chapters teach which skills (image, control, production, video, builder).

---

## The arc in one sentence

You start as a **template user** (run what others built), become a **workflow adjuster** (change prompts, models, and settings), then a **workflow builder** (design pipelines for product, social, and cinematic content).

---

## Chapter progression

| Phase | Chapters | You will… |
|-------|----------|-----------|
| **Orient** | 0 | Pick tools, learn the method, know when to use local vs cloud |
| **First wins** | 1 | Browse inspiration, generate your first image, learn core nodes |
| **Image craft** | 2 | Master txt2img, img2img, tuning, and LoRA |
| **Control** | 3 | Pose, consistency, inpaint, face fix |
| **Production** | 4 | Upscale, product shots, social formats, viral hooks |

## Recommended pace

| Week | Focus | Minimum output |
|------|-------|----------------|
| 1 | Ch. 0–1 | 1 successful template image |
| 2–3 | Ch. 2 | 5-image styled series with LoRA |
| 4 | Ch. 3 | Same character in 3 scenes |
| 5 | Ch. 4 | One upscaled social-ready post |
| 6–7 | Ch. 5 | One 3–5 s motion clip |
| 8+ | Ch. 6 | One reusable signature workflow |

Adjust if you rely on cloud GPUs — budget RunComfy time for Chapters 3–5.

---

## What “viral quality” means here

Not luck — **repeatable craft**:

- Strong composition and lighting (ControlNet + good prompts)
- Consistent characters or brand look (LoRA + reference workflows)
- Resolution that survives social compression (upscale pass)
- Motion that hooks in the first second (video chapter)
- Fast iteration (templates → custom workflow you own)

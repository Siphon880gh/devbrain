## Learning objectives

- Name each tool in the ComfyUI-like ecosystem and its role.
- Know which tools are free vs paid and why GPU hosting costs money.
- Avoid dead or sunsetted services (Kimara.ai is shut down — do not use).

---

## Tool map

### Free workflow browsing and inspiration

| Tool | Role | When to use |
|------|------|-------------|
| [Comfy.org Workflows](https://comfy.org) | Official template library | Study how real workflows connect nodes |
| ComfyUI template sites | Community workflow repos | Find inpaint, upscale, video examples |
| [OpenArt.ai](https://openart.ai) | Style and creative gallery | Mood boards and prompt ideas — not a full ComfyUI replacement |

### Free local options

| Tool | Role | When to use |
|------|------|-------------|
| **Native ComfyUI** | Full open-source node editor | Long-term mastery; needs disk space and ideally NVIDIA GPU |
| **Stability Matrix** | Desktop manager for ComfyUI and peers | Reduce install pain on Windows |
| **InvokeAI** | Simpler local UI | Bridge between single-prompt apps and node thinking |

### Cloud / hosted options

| Tool | Role | When to use |
|------|------|-------------|
| **Promptus.ai** | No-code visual templates (ComfyUI-style) | First runs without local setup |
| **RunComfy** | Hosted ComfyUI + cloud GPU | CUDA workflows, custom nodes, heavy models, video |

---

## Cost reality

- **ComfyUI locally** — free software; you pay with hardware, electricity, and setup time.
- **Promptus / RunComfy** — freemium or credit-based because GPU hours cost money.
- **OpenArt / Comfy.org browsing** — free for inspiration; running workflows still needs a GPU somewhere.

---

## Common-denominator mindset

Learn workflows built on:

- **Checkpoints:** SDXL, Flux (Dev/Schnell), turbo variants like Z-Image-Turbo
- **Core nodes:** Load Checkpoint, CLIP encode, KSampler, VAE decode, Empty Latent
- **Add-ons:** LoRA, ControlNet, upscale nodes

These transfer between local ComfyUI and RunComfy with minimal changes. Promptus exposes many of the same ideas as simplified templates.

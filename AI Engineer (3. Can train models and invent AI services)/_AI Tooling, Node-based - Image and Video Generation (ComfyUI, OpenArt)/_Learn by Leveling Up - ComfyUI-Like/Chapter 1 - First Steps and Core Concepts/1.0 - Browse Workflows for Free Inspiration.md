**Platform note:** Browse only — Comfy.org and OpenArt; no GPU required.

---

## Learning objectives

- Navigate Comfy.org workflow library and read a workflow graph without running it.
- Use OpenArt for prompt and composition ideas.
- Identify core node groups in any template before you execute it.

---

## Steps

### 1. Comfy.org Workflows

1. Go to [comfy.org](https://comfy.org) and open the workflow/template section.
2. Filter or search for **text to image** or **image generation**.
3. Open a workflow preview and locate these groups (names may vary):
   - Load Checkpoint / model loader
   - CLIP Text Encode (positive + negative)
   - Empty Latent Image or latent source
   - KSampler
   - VAE Decode → Save Image
4. Screenshot or bookmark **two** workflows: one minimal, one advanced.

### 2. OpenArt.ai for creative direction

1. Browse styles close to content you want to make (product, portrait, cinematic).
2. Save **prompt phrases** and **aspect ratios** — not necessarily the exact workflow.
3. Treat OpenArt as a mood board; execution stays on ComfyUI / Promptus / RunComfy.

### 3. Read before you run

For each bookmarked workflow, write down:

| Question | Your notes |
|----------|------------|
| Which checkpoint? | |
| Resolution? | |
| Extra branches (LoRA, ControlNet)? | |
| Likely platform (local vs cloud)? | |

---

## What to look for in 2026 templates

- **Flux or SDXL** checkpoints for still images
- **Turbo / Schnell** variants for fast iteration (fewer steps)
- Separate **positive and negative** prompt nodes
- Optional **upscale** branch at the end

---

## Beginner mistakes

- Downloading random workflows without checking required models.
- Assuming OpenArt “Generate” equals your ComfyUI setup.
- Skipping the negative prompt because a preview looked fine once.

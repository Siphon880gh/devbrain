# First Skills to Install in OpenClaw

If you are starting fresh, install a small core toolkit first.

## 1) Web browsing/search

- Real-time internet lookup
- Useful for research and live questions
- Typical names: `browser`, `web_search`, `internet`

## 2) File system access

- Read/write local files
- Useful for outputs, automation, and organization
- Typical names: `filesystem`, `file_manager`, `local_files`

## 3) Terminal/command execution

- Run shell commands
- Useful for coding and automation
- Typical names: `terminal`, `shell`, `command_runner`

## 4) Code execution (Python or Node)

- Run scripts and data processing tasks
- Useful for non-trivial logic and automation
- Typical names: `python`, `code_executor`

## 5) API/HTTP requests

- Connect to external services and SaaS tools
- Useful for integrations (Slack, Notion, DB-adjacent APIs)
- Typical names: `http`, `requests`, `api_client`

## Why these five first

Together they let OpenClaw:

- access information (web)
- store and manage data (files)
- act on the local system (terminal)
- compute and transform data (code execution)
- integrate with external tools (API)

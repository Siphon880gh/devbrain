# OpenClaw TUI Primer

TUI means **Terminal User Interface**.

The TUI connects to the gateway at:

`ws://127.0.0.1:18789`

## Basic Flow

1. Start the gateway:
   `openclaw gateway`
2. Open the TUI:
   `openclaw tui`
3. Type a message and press Enter.

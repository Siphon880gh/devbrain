# TUI Command: `/agent` vs `/agents`

In OpenClaw TUI, these commands are part of the same command family.

| Command | Meaning |
| --- | --- |
| `/agent <id>` | Switch directly to a specific agent ID, such as `/agent main` or `/agent research`. |
| `/agents` | Open the list/picker of available agents. |

If you type only `/agent` with no ID, it falls back to `/agents`.

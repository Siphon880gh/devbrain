When running git to github commands on your terminal, you may be asked to sign in. It opens a web browser page to Github OAuth flow.

If your device is new, you may be asked to enter the device code at the OAuth flow pages. 

When googling where the device code is, people and AI will mislead you into thinking it's on your email or Gmail app. The device code has been outputted at the terminal immediately after you ran `gh auth login` (or whichever command triggered the sign in) and before the web browser opens the page. Look back to the terminal.

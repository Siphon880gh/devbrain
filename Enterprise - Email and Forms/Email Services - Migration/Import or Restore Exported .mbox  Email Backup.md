
After exporting your emails into a single .mbox file (aka mailbox), it is important to verify that the backup completed successfully. You can import the exported files into email clients such as Apple Mail, Thunderbird, or Outlook to confirm the emails are readable and complete.

---

# Viewing an Export in Apple Mail (Mac)

Apple Mail can open `.mbox` exports and is one of the easiest ways to verify a backup.

## Steps

1. Open **Apple Mail**.
2. Go to:

```
File > Import Mailboxes
```

3. Select:

```
Files in mbox format
```

4. Browse to your exported `.mbox` file.
5. Complete the import process.

Apple Mail will create a new **Imported** mailbox section where you can browse the emails.

## Why This Helps

- Confirms the export worked
- Lets you browse emails offline
- Verifies folders and attachments exist
- Provides an additional backup copy

---

# Viewing an Export in Mozilla Thunderbird

Thunderbird supports `.mbox` and `.eml` files.

## Steps

1. Open Thunderbird.
2. Install the add-on:

```
ImportExportTools NG
```

3. Right-click a folder.
4. Choose:

```
ImportExportTools NG
```

5. Select:

```
Import mbox file
```

6. Open your exported mailbox file.

Thunderbird will import the mailbox so you can browse the emails.

## Best For

- Large mailbox verification
- Cross-platform testing
- Advanced mailbox management

---

# Viewing an Export in Microsoft Outlook

If your export is in `.pst` format, Outlook can open it directly.

## Steps

1. Open Outlook.
2. Go to:

```
File > Open & Export > Open Outlook Data File
```

3. Select your `.pst` file.

The mailbox will appear in Outlook’s sidebar for browsing.

There are several ways to export your Hostinger emails depending on how you normally access your mailbox. You can export directly from Hostinger Webmail or use an email client such as Microsoft Outlook, Mozilla Thunderbird, or Apple Mail.

For large mailboxes or full-account backups, using an email client with IMAP is usually the most reliable method.

---

# Method 1: Export Directly from Hostinger Webmail

This is the fastest method if you only need a quick backup of specific folders or messages.

## Steps

1. Log in to your Hostinger account.
2. Open **Emails** and launch **Webmail**.
3. Open the folder you want to export.
4. Select the emails or choose **Select All**.
5. Click the **More** menu or the three-dot menu.
6. Choose **Download**.
7. Select your preferred export format.

## Export Formats

* **.eml** — Saves emails as individual files
* **.mbox** — Saves emails into a mailbox archive format
* **ZIP archive** — Some interfaces package the export into a ZIP file

## Important Note

Some newer Hostinger Webmail interfaces may not show export options. If that happens, switch to the older Webmail interface.

## FAQ

Q: Does it preserve file attachments in the .mbox format?
A: Yes

# Method 2: Export Using Microsoft Outlook

Outlook is one of the best options for exporting large Hostinger mailboxes.

## Steps

1. Open Microsoft Outlook.
2. Add your Hostinger email account using IMAP.
3. Wait for Outlook to fully sync all emails and folders.
4. Go to:

```text
File > Open & Export > Import/Export
```

5. Select:

```text
Export to a file
```

6. Choose:

```text
Outlook Data File (.pst)
```

7. Select your mailbox.
8. Enable **Include subfolders**.
9. Save the `.pst` file.

## Best For

* Large mailboxes
* Full backups
* Windows users
* Long-term archives

# Method 3: Export Using Mozilla Thunderbird

Thunderbird is a free and powerful option for exporting Hostinger emails.

## Steps

1. Install Mozilla Thunderbird.
2. Add your Hostinger email account using IMAP.
3. Wait for all folders to sync.
4. Install the add-on:

```text
ImportExportTools NG
```

5. Right-click the mailbox or folder.
6. Choose:

```text
ImportExportTools NG
```

7. Select:

```text
Export all messages in the folder
```

8. Choose the export format.

## Supported Formats

* `.mbox`
* `.eml`
* HTML
* Plain text

## Best For

* Free backups
* Advanced exporting
* Cross-platform support

---

# Method 4: Export Using Apple Mail (Mac)

Apple Mail is a simple way for Mac users to export Hostinger emails.

## Steps

1. Open **Apple Mail**.
2. Go to:

```text
Mail > Add Account
```

3. Choose:

```text
Other Mail Account
```

4. Enter your Hostinger email credentials.
5. Configure the account using **IMAP**.
6. Wait for all folders to sync.
7. Right-click the mailbox or folder.
8. Select:

```text
Export Mailbox
```

9. Save the exported `.mbox` file.

## Best For

* Mac users
* Simple offline backups
* `.mbox` exports

# Which Export Method Is Best?

| Method            | Best For              |
| ----------------- | --------------------- |
| Hostinger Webmail | Quick manual exports  |
| Outlook           | Full mailbox backups  |
| Thunderbird       | Free advanced exports |
| Apple Mail        | Mac backups           |

For very large mailboxes, Outlook or Thunderbird are usually the safest and most complete export options.

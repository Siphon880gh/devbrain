You simply have to login to the same account that has the original machine(s).

On the new machine, make sure there's tailscale. If there isn't, install Tailscale per [[2. Install Tailscale on Mac and Explore]] or [[2. Install Tailscale on Linux (Debian 12) and Explore]]. Don't worry about creating your own account or not because you can always sign out.

WhatsMyDNS is one of several tools you can use to see whether your new DNS records have propagated across different locations:
[https://www.whatsmydns.net/](https://www.whatsmydns.net/)

**Helpful tip:** You may see that the record has propagated in another state or country, but not yet in your current location. If your VPN lets you choose a specific state or country, you can connect through a location where the update has already propagated and test it right away.
